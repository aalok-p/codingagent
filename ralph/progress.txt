## Codebase Patterns
- Frontend is plain React 19 + Vite + React Router v7 (JSX files, no TypeScript, no Tailwind)
- API modules use separate Axios instances with localStorage-based JWT interceptor
- Toast system via ToastContext + ToastProvider; use `const toast = useToast()` and call `toast(msg)` or `toast(msg, 'error')`
- Styling is custom CSS in App.css with CSS custom properties and `@media (prefers-color-scheme: dark)` for dark mode
- Auth state in localStorage keys: `token`, `userId`, `role`
- Protected routes via `<ProtectedRoute>` wrapper in App.jsx reading localStorage
- All CSS class patterns: `.btn-primary`, `.btn-small`, `.btn-secondary`, `.btn-icon`, `.modal-overlay`, `.modal`, `.modal-actions`, `.form-group`, `.empty-state`, `.page-loading`, `stock-badge`/`stock-high`/`stock-medium`/`stock-low`
- Backend orders already return items with `product_name` in GET /orders response
