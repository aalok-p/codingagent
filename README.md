## What is Ralph Loop?
- It's an autonomous coding agent loop
- Define the project as a set of user stories in ralph/prd.json
- Ralph reads the pRD, picks highest priority unfinished story and implements it.
- repeats until all stories are complete
- each iterations learns from previous ones with ralph/progress.txt
